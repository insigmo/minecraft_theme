Place your custom WAV files here.

Expected files:
- `sfx/tab-switch.wav`
- `sfx/run-start.wav`
- `sfx/debug-start.wav`
- `sfx/run-stop.wav`
- `sfx/debug-stop.wav`
- `sfx/error.wav`
- `music/background.wav`

Current repository contains placeholder files with these names. Replace them with valid WAV audio keeping the same filenames.

If files are missing or invalid, the plugin falls back to a system beep for short SFX, and background music stays disabled.
